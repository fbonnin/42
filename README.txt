Corewar:
Projet réalisé à 3 personnes
Corewar est un jeu dans lequel des programmes s'affrontent dans une arène virtuelle.
L'arène est constituée d'une mémoire circulaire dans laquelle sont placées les instructions des différents programmes.
Les programmes étant exécutés parallèlement dans le même espace mémoire, ils ont la possibilité de s'écrire les uns sur les autres, donnant ansi au jeu une grande  complexité.
Le projet consiste d'une part à coder une machine virtuelle permettant à la partie de se dérouler, et d'autre part un programme qui compile un programme écrit dans un langage type assembleur en bytecode compréhensible par la machine virtuelle.
